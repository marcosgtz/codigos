Bienvenido a mi mini-tutoriales


Ésta página la hice con el propósito de compartirles algunos de mis conocimientos y que en google son algo de tediosos de encontrar. Todo esto fue realizado sobre un Debian 10.

Antes de continuar, necesitas saber al menos estos detalles
  • Existen 2 formas de ejecutar un php (que yo sepa)
    1. Desde un explorador web (chrome, firefox, ie, etc)
    2. Desde una consola de linux o windows
  • Para poder ejecutar codigo php en un explorador web, necesitas tener apache, aunque existen otras alternativas como Nginx o IIS
  • Para integrar php con apache hay 2 formas:
    1. Usando libapache2-mod-php que es el más común, si estas usando XAMPP muy seguramente tienes esto habilitado
    2. Usando php-fpm, que es un servicio muy separado de apache, la ventaja que éste es mucho mejor para páginas con mucho tráfico
  • PHP-CLI es ejecutar código php desde la consola de linux o windows
  • Todo los mini-tutoriales que ves acá está hecho enfocado a apache2
