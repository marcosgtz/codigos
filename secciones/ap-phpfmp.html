Apache con php-fpm


PHP-FPM es una alternativa (mucho mejor y más rápido) al tradicional manejo de procesos entre apache y php. No entraré en detalles, puedes buscar en google las diferencias entre mod_php y fpm. Pero si te enseñaré a instarlarlo

1. Si tienes el apache2-mod_php, hay que deshabilitarlo primero
# a2dismod php7.3 # /etc/init.d/apache2 restart
2. Instalas el paquete de php-fpm
# apt install php-fpm
3. Habilitas la configuracion de php-fpm y el modulo fcgi
# a2enconf php7.3-fpm # a2enmod proxy_fcgi
4. Editas /etc/apache2/conf-enabled/php7.3-fpm.conf
Aprox linea 16: cambias "Require all denied" por "Require all granted"

Esto permitirá ejecutar los php's, aunque las recomendaciones es ponerlo dentro de VirtualHost o Directory

5. Reinicias apache
# /etc/init.d/apache2 restart
